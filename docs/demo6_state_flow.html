Demo6 Certified State Flow

当前 demo6 的符号状态流程：任务 placement states 和 bridge states 如何组成一条 certified route。

24total states
7region placement states

13unique selected bridges
准确说法：除了 HOME / PICK / DROP / RETURN 附近的任务 placement states，其余中间状态都是 bridge states。 连续 bridge states 之间不是任意相连，而是因为它们共享一个 IRIS region。例如 P55 ∩ P9 到 P9 ∩ P46 的转移由公共区域 P9 作为 convex certificate。

State Meaning

Region placement state
状态只属于一个 IRIS region，例如 P55。它表示完整 formation envelope 已经被认证放在 P_i 内部。
Bridge state
状态属于两个区域交集，例如 P55 ∩ P9。它表示 formation 可以站在 overlap 中，用来切换 active region certificate。

Route Segments

HOME to PICKempty, line
出发前从 triangle 切到 line，经过多个 bridge states 到达 PICK。
P55 -> P9 -> P46 -> P67 -> P6 -> P41 -> P3
PICK to DROPloaded, triangle
PICK 处切回 triangle 并拿 payload，载货阶段使用 triangle bridge states。
P3 -> P26 -> P42 -> P58
DROP to RETURNdelivered, line
DROP 后 payload 已经释放，返回阶段切回 line。
P58 -> P39 -> P41 -> P30
RETURN to HOMEdelivered, line
经过 RETURN 后回 HOME，最终在 HOME 切回 triangle。
P30 -> P6 -> P67 -> P46 -> P9 -> P55
