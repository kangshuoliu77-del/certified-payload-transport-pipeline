# Gazebo Pose Replay Plugin

`PayloadPoseReplaySystem.cc` is the C++ Gazebo system plugin used by the
recording demo.

It reads a CSV generated by:

```text
tools/export_gazebo_pose_stream.py
```

and updates all named models inside Gazebo in process. This avoids slow
per-frame `ign service` calls and makes the recording demo smoother.

Build through:

```bash
./scripts/build_gazebo_pose_replay_plugin.sh
```

The main user-facing script builds it automatically:

```bash
./scripts/run_gazebo_recording_lift_demo.sh 8
```
